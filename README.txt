== Description ==

A simple WordPress plugin to display current weather data for any location and doesn't get in your way. Handcrafted with love from Boston, MA by <a href="https://twitter.com/fleetingftw">James Fleeting</a>.

== Installation ==

1. Upload `./simple-weather/` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Configure options through the 'simpleWeather' menu under 'Settings' in WordPress.
